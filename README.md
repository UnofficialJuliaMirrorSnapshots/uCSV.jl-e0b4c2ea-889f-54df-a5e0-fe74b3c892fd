## Getting Started

First, [Install the package](https://docs.julialang.org/en/v1.0/stdlib/Pkg/#Getting-Started-1)
```julia
(v1.0) pkg> add uCSV
```

Then head on over to the [![](https://img.shields.io/badge/docs-latest-blue.svg)](https://cjprybol.github.io/uCSV.jl/latest)!
